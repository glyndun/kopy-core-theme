Button

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Basic usage

Use `.btn` class to instantiate a button element

Predefined button styles

Bootstrap includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.

btn-primary
btn-secondary
btn-success
btn-danger
btn-warning
btn-info
btn-dark
btn-light

Outline buttons

In need of a button, but not the hefty background colors they bring? Replace the default modifier classes with the `btn-outline-{theme}` ones to remove all background images and colors on any button.

btn-outline-primary
btn-outline-secondary
btn-outline-success
btn-outline-danger
btn-outline-warning
btn-outline-info
btn-outline-dark
btn-outline-light

Rounded

Add `btn-rounded` class to make the button rounded.

Rounded outline

You can use `btn-outline-{theme}` and `btn-rounded` together to make the button outline and rounded at the same time.

Button circle

Use `btn-circle` class to make a circle button.

Block button

Create responsive stacks of full-width, “block buttons” like those in Bootstrap 5 with a mix of our display and gap utilities. By using utilities instead of button specific classes, we have much greater control over spacing, alignment, and responsive behaviors.
